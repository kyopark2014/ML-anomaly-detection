# Cloud에서 이상(Anomaly)의 확인

인프라가 디도스(DDOS, 분산 서비스 거부) 공격을 받는 상태인지, 또는 네트워크 액티비티가 계속 증가하고 있는지 등에 대해 자동으로 추적하기 위하여 RCF를 이용하여 이상(Anomaly)를 검출하고자 합니다.
